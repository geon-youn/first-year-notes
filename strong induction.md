# Strong Induction
Let $P(n)$ be a [[predicate]] that depends on an arbitrary positive [[integers|integer]] $n\geq n_0$. If $P(n_0)$ (**base case**) and for all positive integers $n\geq n_0$, if for all positive integers $k$ we assume $1\leq k \leq n,P(k)$ (**induction hypothesis**), then show $P(n+1)$ is true (**inductive step**). Then, for all positive integers $n\geq n_0$, $P(n)$ is true.
