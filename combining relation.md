# Combining relation
Given two [[binary relation|relation]]s $R_1$ and $R_2$, we can combine them using basic [[set]] operations to form new relations like

- $R_1\cup R_2$ [[union]]
- $R_1\cap R_2$ [[intersection|intersect]]
- $R_1-R_2$
- $R_2-R_1$
