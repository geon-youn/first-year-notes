# Proper Subset
If $A\subseteq B$ but $A\neq B$, then $A$ is a proper [[subset]] of $B$. That is, there is some element in $B$ that is not in $A$. 

$$A\subset B \text{ if }(\forall x)(x\in A\rightarrow x\in B)\land (\exists x)(x\in B\land x\notin A)$$
