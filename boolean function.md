# Boolean Function
A **boolean function** is a [[First Year/function]] that takes in `Bool` [[value]]s as [[parameter]]s and returns a `Bool`. All boolean functions can be constructed using just
- `not` and `and`,
- `not` and `or`,
- `nand` (Sheffer stroke), or
- `nor` (Peirce arrow).
