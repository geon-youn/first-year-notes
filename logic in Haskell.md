# Logic in Haskell
A form of **quantifier-free first-order logic** is embedded into Haskell:
- `Bool`, a [[type]] of truth [[value]]s (called [[boolean]]s);
- [[boolean function]]s;
- [[predicate]]s;
- [[conditional expression]]s;
- [[guarded function definition]]s;
- [[case expression]]s.
